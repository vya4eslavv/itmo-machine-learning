# itmo-machine-learning

В данном репозитории представлены решения и датасеты лабораторных работ по курсу машинного обучения в ИТМО.  
Работы охватывают задачи веб-скрейпинга, обучения с учителем и без учителя, обработки изображений и аудио, а также генеративных моделей.

## Веб-скрейпинг и парсинг

Реализован краулер и парсер для сбора данных о LEGO-наборах с сайта BrickEconomy.  
Сформирован датасет с категориальными и числовыми признаками, включая целевую переменную для ML-задач - предсказание средней рыночной цены набора в будущем.
[Парсинг LEGO-наборов и сбор датасета](./parser/lab1_parser.ipynb)

## Обучение с учителем (Supervised Learning)

На основе собранного датасета LEGO-наборов решена задача регрессии: предсказание будущей средней цены набора (`next_avg_price`).  
Проведен EDA, выполнен one-hot encoding категориальных признаков, обучены `Linear Regression` и `Lasso`, а также реализована собственная версия линейной регрессии с подбором гиперпараметров.

[Предсказание будущей цены LEGO-наборов](./lab2/lab2_supervised.ipynb)

## Обучение без учителя (Unsupervised Learning)

Для датасета LEGO-наборов выполнена кластеризация по признакам, связанным с количеством деталей, числом минифигурок и ценой.  
Реализован baseline, обучен `K-Means`, рассчитан silhouette score и дополнительно написана собственная реализация алгоритма кластеризации.

📎 [Кластеризация LEGO-наборов](./lab3/lab3_unsupervised.ipynb)

## Обработка текста (NLP)

Собран корпус стихотворений с сайта Stihi.ru, после чего решена задача классификации автора по тексту произведения.  
Выполнены очистка текста, токенизация, построение словаря, кодирование последовательностей и обучение LSTM-модели для определения автора стихотворения.

[Классификация авторов стихотворений по тексту](./lab4/lab4_text.ipynb)

## Обработка аудио

Решена задача распознавания эмоций по аудиозаписям речи.  
Из аудио извлечены MFCC-признаки, обучена LSTM-модель для классификации эмоций (`neutral`, `happy`, `angry`), а полученные эмбеддинги визуализированы с помощью t-SNE.

[Распознавание эмоций по аудиозаписям речи](./lab6/lab6_audio.ipynb)
